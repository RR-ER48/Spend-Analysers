# Spend Analysers

Spend Analysers is a modern web application designed to help users track, analyze, and manage their expenses efficiently. It provides a simple and interactive interface to monitor spending habits and make better financial decisions.

# 🚀 Features

📊 Track daily expenses
🧾 Add and manage transactions
📈 Visualize spending patterns
🔍 Categorize expenses
⚡ Fast and responsive UI
💡 Simple and user-friendly design

# 🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript, React
Styling: CSS / Sass
Build Tools: Node.js, npm
Version Control: Git & GitHub

🔐 User authentication
☁️ Cloud data storage
